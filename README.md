# Diagonal-Snake
This is a version of the classic video game "Snake", except it has a lot of differences. You can move diagonally, A wider window of error, checkpoints, and bosses! 

# Diagonal Movement
The entire reason for it's name, this version of snake allows you to move diagonally! Surprisingly something I haven't seen anyone do. Press two keys at the same time to see this effect (uses the arrow keys)
